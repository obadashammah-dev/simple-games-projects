# C++ Mini Games Collection

This repository contains two console-based C++ projects developed to practice programming fundamentals, problem-solving, and game logic.

## Projects

### 1. Rock-Paper-Scissors Game

A classic Rock-Paper-Scissors game where the player competes against the computer.

#### Features

* Random computer choices
* Multiple game rounds
* Round-by-round results
* Final game winner determination
* Replay option
* Uses enums, structs, functions, and random number generation

#### Concepts Practiced

* Enumerations (Enums)
* Structures (Structs)
* Functions
* Loops
* Conditional Statements
* Random Number Generation

---

### 2. Math Quiz Game

An interactive math quiz game that generates random arithmetic questions.

#### Features

* Randomly generated questions
* Multiple operation types:

  * Addition
  * Subtraction
  * Multiplication
  * Division
* Difficulty levels
* Score tracking
* Pass/Fail evaluation
* Immediate answer feedback

#### Concepts Practiced

* Data Structures
* Arrays
* Enums
* Functions
* Input Validation
* Random Number Generation
